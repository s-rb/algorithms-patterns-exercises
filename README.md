# Algorithms and Data structures
Simple implementations of some algorithms and data structures

## Algorithms:
* Binary search
* Bubble sort
* Merge sort
* Quick sort
* Rabin-Karp algorithm for searching substring
* Array Max Value (linear) - O(n)

## Data structures:
- Single linked list
- Double linked list
- Binary tree
